## Что такое Git?

Git — это система контроля версий, которая позволяет отслеживать изменения в файлах и работать над проектом в команде.

## Основные команды

- `git init` — создать репозиторий
- `git add` — добавить файлы в отслеживание
- `git commit` — сохранить изменения
- `git push` — отправить на GitHub
- `git pull` — скачать изменения

## Зачем это нужно?

Git незаменим при разработке. Благодаря ему можно:
- Вернуться к любой предыдущей версии
- Работать над новыми фичами в отдельных ветках
- Не бояться что-то сломать
