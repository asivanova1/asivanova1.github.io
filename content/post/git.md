Git — это инструмент, который сохраняет историю изменений в коде.

### Основные команды:
- `git add` — добавить файлы
- `git commit` — сохранить изменения
- `git push` — отправить на GitHub
- `git pull` — скачать изменения

Git незаменим при работе в команде и для сохранения истории проекта.
