Markdown используется для написания постов на этом сайте.

### Синтаксис:
- `# Заголовок 1`
- `## Заголовок 2`
- `**жирный текст**`
- `*курсив*`
- `- пункт списка`
- `[ссылка](url)`

Очень удобно для блогов и документации.
