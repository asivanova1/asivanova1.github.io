## Что такое Markdown?

Markdown — это простой язык разметки, который позволяет форматировать текст без использования мышки.

## Основной синтаксис

```markdown
# Заголовок 1
## Заголовок 2
**жирный текст**
*курсив*
- пункт списка
1. нумерованный список
[ссылка](https://example.com)
`код в строке`
```

## Где используется?

- Посты на этом сайте
- README на GitHub
- Документация

Markdown очень удобен — текст остаётся читаемым даже в исходном виде.
