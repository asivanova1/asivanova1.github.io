Markdown — лёгкий язык разметки

11.04.2026

Markdown — это простой язык разметки, который позволяет форматировать текст.

Синтаксис

  • # Заголовок
  • **жирный текст**
  • *курсив*
  • - пункт списка
  • [ссылка](url)
